# Vipul Avinash — Developer Portfolio

A modern, dark-themed single-page developer portfolio built with React 18, TypeScript, Tailwind CSS v3, and Framer Motion.

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Build for Production

```bash
npm run build
npm run preview
```

## Deploy

Deploy the `dist` folder to Netlify, Vercel, or GitHub Pages.

## Sections

- Navbar (sticky glassmorphism)
- Hero (typewriter effect)
- About
- Skills
- Projects
- Contact
- Footer
